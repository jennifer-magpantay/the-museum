## Status

Finished.

## Run locally

Download or clone the project

```bash
  git clone https://github.com/jennifer-magpantay/the-museum.git
```

After downloading or cloning the file, open it and go to the project's folder

```bash
  cd project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Enjoy it!
